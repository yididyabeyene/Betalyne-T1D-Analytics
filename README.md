# Betalyne-T1D-Analytics
A multi-factor analytical framework for predicting hypoglycemia risk in Type 1 Diabetes patients using real-world continuous glucose monitoring, insulin dosing, and behavioural data from the JAEB Loop Observational Study. Built to inform the evidence base for Betalyne Health, a patient-centred digital health application for T1D self-management.
